How to use

Hold Right Alt anywhere on your computer, speak, then release. The transcript types itself into whatever text field has focus.

Setup

  1. Open C:\dev\voice\.env (create from .env.example if missing).
  2. Paste your OpenAI API key after OPENAI_API_KEY=.
  3. Quit and relaunch the app (tray icon → Quit, then npm start).

Settings

  • CLEANUP_ENABLED=true — LLM polish (punctuation, filler removal). Default on.
  • CLEANUP_MODEL=gpt-4.1-nano — cleanup model. Cheap and fast.
  • OPENAI_REALTIME_MODEL=gpt-realtime-whisper — STT model.

Dependencies

Native modules required for the global hotkey and keystroke typing:

npm install uiohook-napi @nut-tree-fork/nut-js
